# API 키 설정 가이드

보안을 위해 API 키는 `.env` 라는 이름의 파일에 저장해야 합니다.

## 1. `.env` 파일이 보이는 경우
1. 왼쪽 파일 목록에서 `.env` 파일을 클릭하여 엽니다.
2. `API_KEY=YOUR_API_KEY_HERE` 라고 적힌 부분을 찾습니다.
3. `YOUR_API_KEY_HERE`를 지우고, 발급받은 **실제 Gemini API 키(AIza...)**를 붙여넣으세요.
4. 파일을 저장하고 앱을 새로고침(Reload) 하세요.

## 2. `.env` 파일이 보이지 않는 경우
일부 환경에서는 점(`.`)으로 시작하는 파일이 자동으로 숨겨집니다.

1. 파일 목록 상단의 설정이나 메뉴에서 **"숨김 파일 표시(Show Hidden Files)"** 옵션을 찾아보세요.
2. 만약 파일을 찾을 수 없다면, 새 파일을 만들고 이름을 정확히 `.env`로 저장한 뒤 위 1번 내용을 진행하세요.
